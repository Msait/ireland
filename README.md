# Ireland Tours
Pet project by Roman Wyhowski

--
####Requirements
1) [Vagrant] (https://www.vagrantup.com/)
2) [VirtualBox] (https://www.virtualbox.org/)
--
####Instalation
```
$: git clone https://github.com/RobbyWD/ireland.git
```
--
####Run
```shell
$: cd ireland && vagrant up && mvn clean install
```
